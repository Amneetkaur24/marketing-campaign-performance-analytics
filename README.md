# Meta Ad Performance Analytics Dashboard
## Overview

This project analyzes digital advertising performance across multiple campaign types and audience segments. The dashboard provides a centralized view of marketing KPIs to evaluate campaign effectiveness, audience engagement, and return on advertising spend.

The solution enables stakeholders to monitor campaign performance trends, identify high-performing audience segments, and optimize budget allocation across ad formats and platforms.

## Business Objective

Digital marketing teams require clear visibility into campaign performance to optimize budget allocation and improve conversion rates.
This dashboard analyzes advertising performance across campaigns, audience segments, and ad formats to identify high-ROI strategies.

This dashboard was developed to:

• Track campaign performance metrics across multiple dimensions.
• Identify high-performing audience segments.
• Monitor engagement and conversion trends over time.
• Support data-driven marketing decisions.

## Key Insights

• Video ads generated higher engagement but lower conversion rates.
• Certain age segments (25-34) produced the highest purchase conversions.
• Campaigns running during evening hours showed stronger click-through performance.

## Tools & Technologies

• SQL – Data validation and aggregation
• Python – Data preparation and preprocessing
• Microsoft Power BI – Dashboard development and visualization
• DAX – KPI calculations and dynamic metrics
• Data Modeling – Relationship design and performance optimization

## Key Metrics Tracked

• Impressions
• Clicks
• Engagement Rate
• Conversion Rate
• Purchases
• Click Through Rate (CTR)
• Campaign Budget
• Average Budget per Campaign

## Dashboard Capabilities

The dashboard includes several analytical components designed for marketing performance monitoring:

1. Campaign Performance KPIs
High-level indicators providing a snapshot of campaign performance including impressions, engagement, and conversions.

2. Audience Segmentation
Performance breakdown by:
• Gender
• Age groups
• Geographic regions

3. Time-Based Analysis
Campaign trends evaluated across:
• Weekly engagement trends
• Hourly activity distribution
• Monthly campaign performance

4. Ad Format Analysis
Performance comparison across different advertising formats such as:
• Image ads
• Video ads
• Carousel ads
• Stories

5. Interactive Filters
Dynamic filtering allows users to analyze performance by:
• Campaign name
• Platform (Facebook / Instagram)
• Target interest segments
• Selected KPI metrics

## Business Value

This dashboard enables marketing teams to quickly identify performance patterns and optimize advertising strategy by:

• Detecting underperforming campaigns
• Understanding audience engagement behavior
• Improving budget allocation decisions
• Monitoring campaign effectiveness in real time

The interactive design allows stakeholders to explore campaign performance dynamically without requiring technical expertise.

## Project Outcome

The final solution delivers a scalable marketing analytics dashboard capable of supporting campaign performance monitoring and strategic decision-making for digital marketing teams.
